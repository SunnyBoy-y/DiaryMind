# DiaryMind 功能特性详解

## 核心功能

### 1. 智能日记管理
- **多格式支持**: 支持Markdown、DOCX、TXT等多种文档格式
- **智能保存**: 自动保存日记内容，防止数据丢失
- **分类整理**: 按日期自动组织日记文件
- **内容检索**: 快速查找历史日记内容

### 2. AI助手交互
- **智能对话**: 与AI助手进行自然语言对话
- **上下文理解**: AI能够理解日记内容上下文，提供个性化建议
- **流式响应**: 实时显示AI回复，提升交互体验
- **角色定制**: 可自定义AI助手角色和行为

### 3. 语音处理功能
- **语音识别**: 支持多种音频格式转文字（WAV、MP3、M4A、OGG、FLAC）
- **文本转语音**: 将文字内容转换为自然语音
- **多模型支持**: 提供从tiny到large-v3多种识别模型选择
- **多语言支持**: 支持多种语言的语音识别和合成

### 4. 音乐娱乐系统
- **本地音乐播放**: 支持多种音频格式（MP3、WAV、OGG、FLAC、M4A）
- **播放列表管理**: 自动扫描并管理音乐库
- **流式播放**: 支持音乐流式传输，节省内存
- **可视化效果**: 提供音乐频谱可视化动画

### 5. 生活工具集成
- **数字时钟**: 实时显示当前时间
- **待办清单**: 管理日常任务和计划
- **日历视图**: 直观查看日期和安排
- **全屏写作**: 专注模式下的沉浸式写作体验

## 用户界面特色

### 1. 手绘风格设计
- **简约美学**: 采用黑白手绘风格，界面简洁清新
- **卡片式布局**: 所有功能模块以交互式卡片形式呈现
- **响应式设计**: 适配不同屏幕尺寸和设备

### 2. 多视图切换
- **主页视图**: 集成时钟、待办事项和输入区域
- **日记集合**: 展示所有日记文件的列表视图
- **全屏日记**: 专注写作的全屏模式
- **音乐播放器**: 独立的音乐播放界面

### 3. 交互体验
- **快捷键操作**: 支持Ctrl+M快速切换视图
- **动画效果**: 流畅的过渡动画和视觉反馈
- **实时更新**: 界面内容实时响应用户操作

## 技术架构亮点

### 1. 前后端分离
- **前端**: React + Vite + TailwindCSS现代化技术栈
- **后端**: FastAPI高性能Python框架
- **通信**: RESTful API + WebSocket实时通信

### 2. 模块化设计
- **功能解耦**: 各功能模块独立开发和维护
- **API统一**: 所有功能通过标准化API接口提供
- **易于扩展**: 插件化架构便于功能扩展

### 3. 多媒体处理
- **文档处理**: 集成python-docx处理Word文档
- **音频处理**: 整合Whisper ASR进行语音识别
- **语音合成**: 集成火山引擎TTS服务

## 特色功能详解

### AI陪伴对话
DiaryMind的AI助手不仅能够回答问题，还能根据用户的日记内容提供个性化的建议和鼓励。通过分析用户的情绪变化和关注点，AI能够给出贴心的回应，成为用户的智慧伙伴。

### 语音日记
用户可以通过语音录入日记内容，系统会自动将语音转换为文字并保存。这种方式特别适合在行走、运动或其他不方便打字的场景下使用。

### 情绪分析
通过对日记内容的分析，系统能够识别用户的情绪状态，并在界面中以不同的视觉元素展现，帮助用户更好地了解自己的情绪变化。

### 音乐陪伴
内置的音乐播放器不仅可以让用户在写作时享受背景音乐，还能根据日记内容或用户情绪推荐合适的音乐，营造更好的写作氛围。

### 数据可视化
通过图表和日历形式展示用户的写作频率、情绪变化趋势等数据，帮助用户了解自己的写作习惯和心理状态。

## 安全与隐私

### 1. 本地存储
- 日记数据默认存储在本地，保护用户隐私
- 用户完全掌控自己的数据

### 2. 数据加密
- 敏感信息传输采用加密协议
- 本地文件存储安全可靠

### 3. 权限控制
- 明确的功能权限划分
- 最小化权限原则
