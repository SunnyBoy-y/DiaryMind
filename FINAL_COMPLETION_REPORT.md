## 🎯 核心功能说明

### 1. 🔍 高级搜索 (DiarySearch.jsx)
```
支持维度：
- 关键词搜索
- 心情筛选（6种）
- 天气筛选（5种）
- 标签筛选（动态）
- 日期范围筛选
- 多条件组合

使用场景：
- 快速定位特定日记
- 按心情查看日记
- 统计特定时期内容
```

### 2. 📊 统计分析 (DiaryAnalytics.jsx)
```
展示维度：
- 总日记数、总字数
- 写作频率和天数
- 日均字数统计
- 心情分布图表
- 天气分布图表
- 热门标签云
- 智能洞察建议

时间范围：
- 近7天 / 近30天 / 近1年 / 全部时间
```

### 3. 🏷️ 标签管理 (DiaryTagManager.jsx)
```
功能模块：
- 标签管理（创建、编辑、删除）
- 颜色自定义（10种配色）
- 标签云展示
- 分类浏览（8个默认分类）
- 分类统计和进度

命名规范：
- 标签：#tagname
- 分类：[分类名]
```

### 4. 🎵 音乐播放器 (AdvancedMusicPlayer.jsx)
```
控制功能：
- 播放/暂停/前进/后退
- 进度条调节
- 音量控制（0-100%）
- 循环模式（3种：无/全部/单曲）
- 随机播放
- 播放列表搜索

UI特性：
- 专辑封面展示
- 实时播放指示
- 列表管理
```

### 5. 📤 导出分享 (DiaryExportShare.jsx)
```
导出格式：
- Markdown (.md)
- HTML (.html)
- 纯文本 (.txt)
- 打印输出

分享功能：
- 生成分享链接
- 一键复制
- 批量导出（开发中）

统计信息：
- 导出文件数
- 文件总大小
- 导出格式统计
```

### 6. 🎨 主题系统 (ThemeProvider.jsx)
```
主题选项：
- 浅色 (蓝色系) - 专业感
- 暖色 (橙色系) - 温暖感  
- 清爽 (青色系) - 清晰感
- 自然 (绿色系) - 舒适感

模式切换：
- 浅色模式 ☀️
- 深色模式 🌙

持久化：
- LocalStorage保存用户偏好
```

### 7. ✏️ 增强编辑器 (EnhancedMarkdownEditor.jsx)
```
格式化工具：
- 粗体、斜体、代码
- 标题、列表、链接
- 代码块、引用、表格

快捷键支持：
- Ctrl+B: 粗体
- Ctrl+I: 斜体
- Ctrl+Z: 撤销
- Ctrl+Y: 重做
- Tab: 自动缩进

高级功能：
- 撤销/重做栈
- 分屏实时预览
- 字符/行数统计
- 全屏编辑模式
- 自动保存提示
```
